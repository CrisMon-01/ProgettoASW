### Ambiente di esecuzione 

L'applicazione è stata ideata per essere eseguita nell'ambiente [workstation](environments/workstation/). 
Vanno però utilizzate più finestre (terminali) diverse. In genere, una per l'applicazione e una per il suo client.  

### Esecuzione

Alcune osservazioni: 
* le applicazioni funzionano correttamente anche in presenza di più istanze di ciascun servizio; visto il limitato
numero di nomi a disposizione per ciascuno, potrebbe verificarsi la ripetizione dello stesso nome per differenti
istanze di uno stesso servizio;
* in particolare in presenza di più istanze, può occorrere qualche minuto affinché siano tutte registrate correttamente;
pertanto, qualora inizialmente le richieste del client non ottenessero risposta, si attenda
